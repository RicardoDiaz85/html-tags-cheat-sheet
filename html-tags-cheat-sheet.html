Metadata

The <!DOCTYPE html> declaration is a preamble required at the beginning of every HTML document. It tells the browser which version of HTML the page is written in, ensuring it's rendered in standards mode, which follows the latest web specifications. Without it, browsers might enter "quirks mode," leading to inconsistent rendering and potential layout issues.

The <html> is the root element that encloses the entire HTML document. It follows the DOCTYPE declaration and contains both the code <head> and <body> sections.

The <head> element serves as a container for metadata, which is information about the document itself. This information is not displayed on the web page but is crucial for the browser and search engines.

The <title> element defines the title of the HTML document, which appears in the browser tab and is often used by search engines.

The <meta> element is used for various metadata, such as character encoding, page description, keywords, author information, and viewport settings.

The <link> element is primarily used to link external resources, most commonly stylesheets (CSS).

The <script> element is used to include or reference client-side scripts, typically JavaScript, which can add interactivity to the page.

The <base> specifies a base URL and/or target for all relative URLs in the document.

The <style> allows for embedding CSS styles directly within the HTML document.

Headings

Headings from h1 to h6 identify blocks and sections. Every page must have one h1. Google's web-crawlers use the h1 for search results and SEO.

Paragraphs

Paragraphs of long-form text containing complete sentences go in the <p> (paragraph) tag. Paragraphs help organize and structure your page's content for both readers and search engines. <p> tags are meant for paragraphs and automatically add spacing between them. <br> should not be used between paragraphsâ€”it's for line breaks within a paragraph, not between paragraphs. Using proper tags is better for accessibility, structure, and SEO.

Lists

Types of lists:

  • Unordered lists (<ul>)
  • Ordered lists (<ol>)

Unordered Lists

Unordered lists are marked with bullet points and go in <ul> tags. Each list item goes in an <li> tag.

Ordered Lists

Ordered lists are numbered and go in <ol> tags. Each list item goes in an <li> tag.

Anchors (Links)

Anchor tags create links to other pages by anchoring the URL to some text on the page. The URL goes in the href attribute. Go to DuckDuckGo to search for more information.

Setting target="_blank" will open the link in a new window. See MDN for more details.

Anchor tags can also create links to a new location on the same page. Use the id of the element you want to navigate to, e.g. Jump to the top of the page.

Text Emphasis

Italic

 The <em> element adds extra emphasis to the text within the opening and closing tags. This shows up as italic text in most browsers.

Bold

 You can also use the <strong> element to emphasize text. This shows up as bold text in most browsers.
